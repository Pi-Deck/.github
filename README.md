# What is Pi-Deck?

Pi-Deck is a modern fork of the original **Stream-Pi** project.

While Stream-Pi laid an excellent foundation for Raspberry Pi powered macro decks, Pi-Deck aims to evolve that vision with a modern interface, improved performance, better reliability and a significantly expanded feature set.

Our goal isn't simply to maintain Stream-Pi.

Our goal is to build the best open-source desktop command console available.

---

# Why Pi-Deck?

Modern hardware deserves modern software.

Pi-Deck focuses on:

- Modern UI/UX
- Better Raspberry Pi support
- Faster performance
- Improved stability
- Better plugin support
- Improved developer experience
- Active development
- Community driven features

Whether you're:
